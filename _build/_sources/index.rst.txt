Welcome to the documentation page for Splotch
=============================================

**Simple PLOTs, Contours and Histograms** (Splotch) is a small package with wrapper functions designed to simplify plotting calls from matplotlib.

Splotch able to be installed using `>pip install splotch`, though you can get the lastest stable version using `>pip install git+https://github.com/MBravoS/splotch.git@master`, or the latest development version using `>pip install git+https://github.com/MBravoS/splotch.git@devel`.
